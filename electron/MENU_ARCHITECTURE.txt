                           QUICK REFERENCE
================================================================================

ADD NEW MENU ITEM:
1. Edit electron/main.js
2. Find createApplicationMenu()
3. Add to appropriate submenu:
   {
     label: 'My Action',
     accelerator: 'CmdOrCtrl+Key',
     click: () => {
       mainWindow.webContents.send('menu-action', { action: 'my-action' });
     }
   }
4. Handle in renderer:
   window.api.onMenuAction((data) => {
     if (data.action === 'my-action') { /* ... */ }
   });

HANDLE MENU ACTION:
window.api.onMenuAction((data) => {
  switch (data.action) {
    case 'action-name':
      yourFunction();
      break;
  }
});

DEBUG MENU ACTIONS:
console.log('Menu action received:', data.action);
